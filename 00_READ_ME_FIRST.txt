üåê STEP 3: DEPLOY TO GITHUB (FREE HOSTING)
================================================================================

1. Create GitHub account (if you don't have one): github.com/signup

2. Go to: github.com/new
   - Name: portfolio
   - Choose "Public"
   - Click "Create repository"

3. In your terminal, run these commands:
   git init
   git add .
   git commit -m "Initial portfolio"
   git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
   git branch -M main
   git push -u origin main

   Replace YOUR_USERNAME with your actual GitHub username

4. Go back to GitHub repo ‚Üí Settings ‚Üí Pages
   - Select "Deploy from a branch"
   - Choose "main" branch, "/" folder
   - Click Save

5. Your portfolio is live at:
   https://YOUR_USERNAME.github.io/portfolio/
